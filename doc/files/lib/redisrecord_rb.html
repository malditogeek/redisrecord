RedisRecord

A "virtual" Object Relational Mapper on top of Redis.

This is just a proof-of-concept. Allows you to create relationships between classes and store them in Redis, a key-value storage.

Main repository

github.com/slacker/redisrecord/tree/master

Author

Mauro Pompilio <hackers.are.rockstars@gmail.com>

MIT

Example

 class User < RedisRecord
   database 15
   has_many :posts
 end

 class Post < RedisRecord
   database 15
   belongs_to :user
   has_many :comments
 end

 class Comment < RedisRecord
   database 15
   belongs_to :post
   belongs_to :user
 end
